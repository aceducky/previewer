Preview and edit html code in an easy and simple way. <br/>
Live site: [https://aceducky.github.io/previewer](https://aceducky.github.io/previewer)

Want to test a snippet of html, css or js code or even libraries (using cdn links) without opening an online editor or creating a file? You can use the live site, or download the **[index.html](index.html)** file and open it locally. After opening it, you can bookmark the file for quick access anytime.

* Single file (index.html)
* Easy to use
* Use the live site, or download and bookmark the file for quick access
